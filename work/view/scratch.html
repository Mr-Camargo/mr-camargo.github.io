JavaScript 2017 - 2020 Open Source

Scratch is an open-source, JavaScript-based learning platform for kids that enables them to learn the fundamentals of coding using drag-and-drop blocks.

When I was in elementary school, I started to get curiosity about computer science and the videogame industry. Scratch enabled me to learn basic Javascript algorithms, principles of UI/UX design, and storytelling abilities that I would later use in many of my works and projects.

I started to use Scratch in the classroom, and that propelled me to build more advanced projects.
